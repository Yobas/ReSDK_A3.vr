## Быстрый старт

Если вы новый разработчик:

1. Начните с [Обзора проекта](01_OVERVIEW.md)
2. Изучите [Структуру проекта](02_PROJECT_STRUCTURE.md)
3. Ознакомьтесь с [Особенностями синтаксиса](03_SYNTAX_GUIDE.md)
4. Прочитайте [Модульную систему](04_MODULE_SYSTEM.md)
5. Изучите [Процессы разработки](08_DEVELOPMENT_WORKFLOW.md)

Для опытных разработчиков используйте [Быстрый справочник](QUICK_REFERENCE.md) для быстрого доступа к информации.
