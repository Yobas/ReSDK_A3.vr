# Работа системы

Когда игрок входит в зону с работающей рацией\динамиком (далее "радио") оно регистрирует стрим в словарь всех клиентских радио по указателю.

Когда другой клиент говорит в рацию - на сервере проставляется флаг - клиент с таким-то именем вещает в такую-то частоту.
В этот момент локальный клиент проверяет частотные каналы. Радио на частотах которые он может слышать в данный момент помечаются ретранслятором. При создании ретранслятора создается чанел группа, в которую будут подмешиваться аудио других клиентов.
